# eco-friendly

A lightweight HTML, CSS, and JavaScript dashboard for tracking daily eco-friendly
activities and awarding green points.

## Features

- Add common activities: used bicycle, saved electricity, recycled waste, and used
  reusable bag.
- Add a custom eco-friendly activity.
- Award green points in JavaScript without a backend.
- Update the total points and daily activity list without reloading the page.
- Show empty, validation, and populated states.

## Run

Open `index.html` in a browser.
